## рҹҡҖ Quick build instructions (30 min)

1. **(5 min)** Create a blank Notion page вҶ’ title "AI Stack Dashboard"
2. **(15 min)** Create the 5 databases in order below (each as "Full page database")
3. **(5 min)** On the dashboard page, add 4 "Linked view of database" to show the right views in the right place
4. **(5 min)** Add static content blocks (rituals, shortcuts, notes)

Sections below give you **every property and every view** for each database.

# рҹ“… Rituals вҖ” checklist contents

## Monday (15 min) вҖ” weekly planning
- [ ] Open `Projects вҶ’ Timeline` and identify this week's 3 deadlines
- [ ] Open `Clients вҶ’ To re-engage` and list the 3 priority messages
- [ ] Open `Automation Backlog вҶ’ Top priorities` and pick 1 workflow to push this week
- [ ] Block time slots in Google Calendar

## Friday (20 min) вҖ” retro
- [ ] Mark delivered projects as вң… Delivered
- [ ] Mark paid projects as `Paid = true`
- [ ] Add a short post-mortem on each delivered project (3 lines max)
- [ ] Log in `Prompt Library вҶ’ Notes / variants` the prompts you actually used this week

## 1st of month (30 min) вҖ” stack review
- [ ] Open `AI Tools вҶ’ Review queue` and process each row: `Last reviewed = today`
- [ ] Identify 1 tool to churn (if `Status = рҹҹЎ Testing` and rarely used)
- [ ] Compute total cost of active stack вҶ’ compare to last month
- [ ] Publish 1 piece of content about your current stack (transparency = easy content)

# рҹҺҜ Usage tips

- **Don't multiply databases.** 5 is the right number. Beyond that it becomes a graveyard.
- **Filtered views > extra databases.** Instead of a "Leads" db, filter `Clients` by `Status = рҹ‘Җ Prospect`.
- **Rollups are your friend.** `Total invoiced` per client auto-calculates. Zero double entry.
- **Bidirectional relations:** verify each relation also appears in the target db (e.g. `Projects вҶ’ Tools used` must show up from `AI Tools вҶ’ Used on project`).
- **Mobile capture:** on each db, create a "Quick capture" view with only `Name + Status + Notes` to add items in 5 seconds from your phone.
