How to use — 50 Prompts to Automate Your Business

The pack I use daily in my own workflows. — Manu, Taiyka

What's in the pack

  • prompts-fr.md — the 50 prompts in French (in case you work with FR clients)
  • prompts-en.md — the 50 prompts in English, organized by category
  • prompts.csv — ready-to-import file for Notion (FR + EN in the same rows)
  • This guide — how to use, import, and remix

The 5 categories

  1. Content — hooks, scripts, captions, carousels (10 prompts)
  2. Sales — cold emails, DMs, pitches, follow-ups (10 prompts)
  3. Operations — SOPs, workflow audits, planning (10 prompts)
  4. Research — syntheses, competitive watch, personas, comparisons (10 prompts)
  5. Customer Support — replies, FAQ, onboarding, refunds (10 prompts)

How to use a prompt

  1. Copy the prompt from the .md or .csv
  2. Replace the {variables} between braces with your info
  3. Paste in your tool: Claude, ChatGPT, Gemini, or an n8n node
  4. Refine the result using the "Tweak this" tip at the bottom of each prompt

Tip: for long prompts, keep a stripped version for mobile use and the full version for focused work sessions.

Import the CSV into Notion

  1. Create a new Notion page → Import → CSV
  2. Select prompts.csv
  3. Notion auto-creates a database with columns: title_fr, title_en, category, prompt_fr, prompt_en
  4. Optional: change category to a Select property for easier filtering

Useful views to create:

  • 📋 List by category — Gallery view grouped by category
  • 🔥 My favorites — add a Checkbox "Favorite" property + filtered view
  • 🌍 FR only / EN only — depending on what you use most

Import into Claude Projects or ChatGPT (memory)

You can upload the .md directly as a reference file in:

  • Claude Projects (Knowledge files)
  • ChatGPT Custom GPT (Knowledge)
  • Gemini Gems (Instructions or attached files)

Then ask: "Use prompt #12 for this situation: [context]" — the AI pulls straight from the pack.

How to remix a prompt

Prompts are starting points, not dogma. Three ways to adapt them:

1. Change the tone

Replace "confident direct ‘you’ voice" with whatever you need (formal, humorous, B2B corporate, etc.).

2. Change output language

All prompts are neutral on output language. Just add "Reply in {language}" at the end.

3. Combine two prompts

Example: prompt #2 (Reel script) + prompt #7 (client storytelling) = a Reel script based on an anonymized client project.

Tips to make your prompts even better

  • More context = better output. Always add: your industry, audience, concrete examples.
  • Add counter-examples. "NEVER X" is as powerful as "do Y".
  • Ask for multiple variants. "Generate 3 versions" forces broader exploration.
  • Keep a personal ‘best prompts’ file where you copy your remixed versions that actually work for you.

Recommended weekly ritual

Block 30 minutes Sunday evening to:

  1. Pick 3 prompts you'll use this week
  2. Remix them with your context (business, client, current project)
  3. Save them as favorites

In 4 weeks, you'll have an arsenal of 12 prompts perfectly calibrated to YOU.
