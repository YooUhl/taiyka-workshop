# Taiyka Brand Assets

Single source of truth for visual identity across web, products, social.

## Files

- [tokens.json](tokens.json) — colors, typography, radii, shadows, spacing. Consume from `web/` (Tailwind config) and product PDFs.
- `logo/` — logo files (TODO: add SVG + PNG variants)
- `fonts/` — self-hosted font files if needed (currently using system Inter)

## Identity

- **Logo concept:** electric blue glowing gear on near-black background
- **Direction:** dark, techy, electric blue/cyan accents, bold white text
- **Voice:** confident, direct, slightly provocative, grounded in real outcomes (see `content-creator/style-guide.md`)

## Color Quick Reference

| Token | Hex | Use |
|---|---|---|
| Electric Blue | `#00A6FF` | Primary brand, CTAs |
| Cyan | `#00E5FF` | Accents, gradient end |
| Navy | `#0A1628` | Backgrounds |
| White | `#FFFFFF` | Text on dark |
