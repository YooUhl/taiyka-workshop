# @manu_ai.to — Système de contenu piloté par IA

> Mon propre pipeline de contenu : scripts, carrousels, captions — générés dans ma voix, pour une audience FR tech / automatisation.

## Le problème
Créer du contenu constant pour @manu_ai.to tout en livrant des projets clients, c'est un combat. J'ai besoin de tenir le rythme sans sonner comme un bot ni copier des templates génériques. Ma voix (confiante, directe, curiosité-first) est spécifique — pas question de la diluer.

## La solution
J'ai construit un système de contenu basé sur des skills Claude Code personnalisées. Un style-guide tiré de mes 20 meilleurs posts alimente un content-creator skill qui génère scripts vidéo, carrousels (méthode 555) et captions dans ma voix. Les hooks viennent de patterns confirmés (curiosité, challenge, observation marché), les closes évitent les CTAs classiques, et chaque sortie inclut des notes de réalisation pour le tournage. Tout est versionné dans mon dépôt de scripts personnel et appelable à la demande.

## Stack
`Claude Code` · `Anthropic API` · `Skills personnalisées (.md)` · `Style guide scrappé IG` · `n8n`

## Résultats
- Réduit le temps de création d'un script de 1h à ~10 minutes
- Audience niche FR AI/automatisation en croissance constante
- Voice-match vérifiable — les skills citent les posts de référence sur lesquels elles se basent
- Système réutilisable : la même approche a été packagée pour des clients

## Statut
Live, utilisé chaque semaine.
